For our JavaScript code, we auto generate source documentation with the powerful, and extensive, jsDoc3.

Features

SassDoc & jsDoc both parse our source files to grab documentation-specific comments. From there, the apps build data trees that get enhanced and filtered before being passed to the views. So we end up with fully styled HTML documents that are easily readable and navigable.

Source file comments

Write documentation comments in the same file as your code.

Gulp integration

Our docsTask() compiles documentation automatically.

Extendedable with plugins

Both apps are plugin-friendly; we can extend our compiled views as needed.
